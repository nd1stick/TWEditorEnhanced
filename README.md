This is The Witcher Editor Enhanced written in Python (tested only on Windows 10). If you're having trouble with the original Java version, give this a try. Use at your own discretion and make sure to BACKUP any saved game prior to editing.

Experimental features:
- Corrupted save recovery, open your saved game file then save and close your saved game file.
- Quest reset capability, go to Quests tab to select the related quest and tick the 'Reset' checkbox.
- Add single items to your inventory or a stack at a time.

To run simply double-click \_\_main\_\_.py
